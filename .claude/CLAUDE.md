# pink-orchestrator

System tray manager for pink-tools services.

```bash
/Users/pink-tools/pink-orchestrator/pink-orchestrator                             # Start in system tray
/Users/pink-tools/pink-orchestrator/pink-orchestrator --health                    # Check health
/Users/pink-tools/pink-orchestrator/pink-orchestrator --version                   # Show version
/Users/pink-tools/pink-orchestrator/pink-orchestrator --update-all                # Update all installed services
/Users/pink-tools/pink-orchestrator/pink-orchestrator --service-update <name>     # Update a service
/Users/pink-tools/pink-orchestrator/pink-orchestrator --service-restart <name>    # Restart a service
/Users/pink-tools/pink-orchestrator/pink-orchestrator --service-stop <name>       # Stop a service
/Users/pink-tools/pink-orchestrator/pink-orchestrator --service-start <name>      # Start a service
```
