# pink-orchestrator

System tray manager for pink-tools services.

## Binaries

- `pink-orchestrator-darwin-arm64` - macOS ARM64
- `pink-orchestrator-linux-amd64` - Linux x64
- `pink-orchestrator-windows-amd64.exe` - Windows x64

## Features

- Install services from GitHub Releases
- Start/stop/restart daemons
- Edit .env configuration
- Auto-restore services on restart
- Plugin API for menu extensions
